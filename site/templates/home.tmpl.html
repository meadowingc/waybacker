Waybacker!

Hi! You've stumbled on Waybacker. This is a service that I'm running that will crawl each of the websites below and, if there are any changes since last crawl, it will request the Wayback Machine to archive them. I mainly made this as way to ensure that if I ever unkowingly break any links in my blog then people can still access the content via these backups.

If you want me to add (or remove) your website to this service, so that it is periodically archived to the Wayback Machine then feel free to send me an email!

This project is open source, you can find the source code here.
